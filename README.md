# FastAPI Microservice + API Autotests

## Описание проекта
Этот проект включает в себя:
* Микросервис на Python с использованием FastAPI
* Тесты API

##  Стек технологий

- Python 3.12+
- FastAPI
- Uvicorn
- Pytest
- Allure
- Requests
- Pydantic
- SQLModel

## Как запустить микросервис

`poetry run fastapi dev src/main.py`

## Как запустить тесты

`poetry run pytest`
